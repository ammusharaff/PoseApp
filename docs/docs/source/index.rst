=====================================
PoseApp Documentation (v2.0)
=====================================

Welcome to **PoseApp** — a modular, AI-assisted posture and activity tracking platform.
Built with PySide6 and OpenCV, it provides real-time motion analytics,
guided exercise scoring, and session-based performance tracking.
